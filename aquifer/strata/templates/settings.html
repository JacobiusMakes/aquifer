CLI Quick Start

Use these commands with your API key to sync files from the command line. 

# Install
pip install aquifer

# De-identify a file locally
aquifer deid clinical_note.txt --vault vault.aqv --password mypass

# Sync local vault to cloud
aquifer vault sync --api-key aq_YOUR_KEY --server {{ server_url }}

# Push files to cloud
aquifer vault push --api-key aq_YOUR_KEY --server {{ server_url }}
